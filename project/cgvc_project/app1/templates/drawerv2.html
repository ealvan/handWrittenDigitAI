Neural Networks

Esté es una página web orientada a niños, tiene por objetivo hacer aprender a los niños los números de 0 al 9. Pueden dibujar con el mouse, y una Red Neuronal Convolucional adivinará que número escribieron.

El juego debe tiene el propósito de ser divertido, para que los niños aprendan jugando.

  1. X: limpiar la pizarra de dibujo.
  2. S: guardar la pizarra de dibujo.
  3. P: predecir la pizarra de dibujo.

Materiales

Los materiales usados son:

  1. Dataset MNIST de Tensorflow
  2. Google Collaboratory para construir el modelo
  3. Lenguaje de programación Python
  4. Para las interfaces se uso:
    1. CSS
    2. JavaScript
    3. HTML
    4. Bootstrap
